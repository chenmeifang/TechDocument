# 2 冒泡排序

谁大谁往右移动

**第一版代码**

```javascript
// 谁大谁往右交换
const arr = [3, 2, 5, 4, 3, 6]
for (let i = arr.length; i > 0; i--) {
    for (let j = 0; j < i; j++) {
        if (arr[j] > arr[j + 1]) {
            swap(j, j + 1);
        }
    }
}
```

**第二版代码：修改了i的初始值**

```js
for (let i = arr.length - 1; i > 0; i--) {
    for (let j = 0; j < i; j++) {
        if (arr[j] > arr[j + 1]) {
            swap(j, j + 1);
        }
    }
}

function swap(arr, i, j) {
    arr[i] = arr[i] ^ arr[j];
    arr[j] = arr[i] ^ arr[j];
    arr[i] = arr[i] ^ arr[j];
}
```



异或运算：相同为0，不同为1。还可以理解为**无进位相加**
